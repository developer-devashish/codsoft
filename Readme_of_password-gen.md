# Password Generator (Tkinter GUI)

A simple and stylish Password Generator application built with Python and Tkinter. This tool helps users create secure passwords with options like length, numbers, symbols, and even only numeric passwords.

---

## Features

* Generate random secure passwords
* Choose password length
* Include:

  * Letters (default)
  * Numbers
  * Symbols
* Option for Only Numbers
* Copy password to clipboard
* Input validation with warning messages
* Modern dark-themed UI

---

## Technologies Used

* Python
* Tkinter (for GUI)
* Built-in libraries:

  * `random`
  * `string`

---

## Installation

1. Make sure Python is installed (Python 3.x recommended)

2. Clone this repository or download the files:

```bash
git clone https://github.com/developer-devashish/codsoft/blob/main/password-gen.py
```

3. Navigate to the project folder:

```bash
cd password-generator
```

4. Run the application:

```bash
python password-gen.py
```

---

## How It Works

1. Enter the desired password length
2. Select options:

   * Include Numbers
   * Include Symbols
   * Only Numbers (optional)
3. Click Generate Password
4. Copy it using Copy to Clipboard

---

## Notes

* If "Only Numbers" is selected, it overrides all other options
* Password length must be greater than 0
* Invalid input will show warning messages
